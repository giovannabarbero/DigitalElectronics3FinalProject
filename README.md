# DigitalElectronics3FinalProject
This repository is going to be used to do the final project for Digital Electronics 3. on LPC1769.
The goal of this project is to make an ambience sound reader. The board is going to get the readings through a microphone module and then show them in dB in the computer. In addition, it's also going to show the sound levels on a 8x8 LED matrix.
